# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-01

### Added
- Initial public release
- `create-vm.sh` to provision a small headless x86 VM on Proxmox
- `postinstall-dev.sh` for baseline dev tooling inside guest
- `set-console-mode.sh` for noVNC/console usability
- Architecture, troubleshooting, and example config docs
- CI workflow for shell and markdown checks
